# inbox/

Drop briefs here. Anything works:

- Text (`.md`, `.txt`)
- Documents (`.pdf`, `.docx`)
- Data (`.csv`, `.json`, `.xlsx`)
- Images / wireframes
- Voice memo transcripts
- One-liners — even a sentence is enough

Each session, Nordlicht Digital scans this folder. New files become tickets. Replies to outstanding questions should be named `REPLY-EMAIL-####.<ext>`.

The firm clears this folder during triage. Originals are preserved at `workspace/tickets/<id>/00-intake/`.
