# WCAG 2.2 AA checklist (excerpt)

- Contrast ≥ 4.5:1 (3:1 for large text)
- Visible focus indicator on every interactive element
- Skip-link to main content
- Labels for every form field
- Keyboard reachable for every interactive element
- No motion if `prefers-reduced-motion`
- Reflow at 320 CSS px without horizontal scroll
- Touch targets ≥ 44×44 CSS px
