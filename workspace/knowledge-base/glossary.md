# Glossary

- **Gate A** — Sandbox → Staging gates (A1 QA, A2 Security, A3 Quality).
- **Gate B** — Staging → Production gates (B1 Regression, B2 Privacy, B3 Compliance, B4 UX Acceptance).
- **DoR / DoD** — Definition of Ready / Done.
- **WIP** — Work-in-progress.
- **Drift** — Any mismatch between expected and observed state (esp. production hashes).
